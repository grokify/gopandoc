## [v0.4.0] - 2026-05-11

### Highlights

- New `citation` package for bibliographic references with multiple citation styles (Simple, MLA, APA, Chicago)
- New `citegen` CLI tool for generating formatted reference lists from JSON

### Added

- `citation` package with `Reference` and `Author` types for bibliographic data ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `Collection` type for loading/saving references from JSON with validation ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `SimpleFormatter` for readable citation output ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `MLAFormatter` for MLA 9th edition citation style ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `APAFormatter` for APA 7th edition citation style ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `ChicagoFormatter` for Chicago Manual of Style (bibliography and notes) ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `FormatMarkdown()` and `FormatMarkdownGrouped()` for markdown output ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- `citegen` CLI tool using Cobra with `generate`, `validate`, and `tags` subcommands ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
- Tag-based filtering and grouping of references ([`bdbf8b9`](https://github.com/grokify/gopandoc/commit/bdbf8b9))
